## ⚡ Installatie & Gebruik
```
# Cloning the repository
git clone https://github.com/JustinLung/the-web-is-for-everyone-scrollbook.git

# Moving to the folder
cd the-web-is-for-everyone-scrollbook

# Install NPM packages
npm install

# Run local server
npm run start
```
